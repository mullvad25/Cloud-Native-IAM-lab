# Tenant Foundation

## Naming Conventions

### Users

Format:
[firstname.lastname@tenant.onmicrosoft.com](mailto:firstname.lastname@tenant.onmicrosoft.com)

Examples:

* alice.hr@
* bob.finance@
* charlie.it@

---

### Admin Accounts

Format:
role-admin@

Examples:

* iam-admin@
* sec-admin@
* intune-admin@
* reader-admin@

---

### Groups

Format:
SG-[Department]-[Purpose]

Examples:

* SG-HR
* SG-Finance
* SG-IT
* SG-Sales
* SG-All-Employees
* SG-Privileged-Eligible

---

### Test Accounts

Format:
test-[purpose]@

Examples:

* test-risk@
* test-device@
